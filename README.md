## Project Idea
The idea is for a small study helper app that shows beginner JavaScript skills without building a full application.

1. **Values, Data Types, and Operations**
2. **Stringing Characters Together**
3. **Control Structures and Logic**
4. **Building Arrays**
5. **Using Arrays**
6. **Creating and Using Objects**
